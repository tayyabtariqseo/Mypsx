# 📊 PSX Sentinel AI Stock Analyzer

Professional-grade technical analysis and AI-driven insights for the Pakistan Stock Exchange (PSX), powered by Google Gemini.

## 🚀 Live Demo
**[mttpsx.streamlit.app](https://mttpsx.streamlit.app/)**

## ✨ Features
- **Live Price Tracking:** Real-time data fetching from the PSX Data Portal.
- **Advanced Technicals:** 11+ indicators including RSI, MACD, ADX, Chaikin, and SuperTrend.
- **Smart Pivots:** Automated Traditional and Fibonacci Pivot Points (calculated from previous session data).
- **AI Analyst Report:** Professional risk/reward assessments and strategy recommendations generated by Gemini-2.0-Flash.
- **Split-Adjusted Data:** Built-in logic to handle historical stock splits (e.g., Systems Ltd 5-for-1 split).
- **Mobile Optimized:** Fully responsive dashboard for analysis on the go.

## 🛠️ Tech Stack
- **Dashboard:** Streamlit
- **Data:** `psxdata`, `yfinance`
- **Technicals:** `pandas-ta`
- **AI Engine:** Google GenAI (Gemini)
- **Visuals:** Plotly Interactive Charts

## ⚙️ Local Setup
1. **Clone the repo:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/psx-sentinel-ai.git
   cd psx-sentinel-ai
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Set up environment:**
   Create a `.env` file and add your Google API Key:
   ```env
   GOOGLE_API_KEY=your_api_key_here
   ```
4. **Run the app:**
   ```bash
   streamlit run app.py
   ```

## 🌐 Deployment
This app is designed to be deployed on **Streamlit Community Cloud**. 
1. Push this code to GitHub.
2. Connect your repo to Streamlit Cloud.
3. Add `GOOGLE_API_KEY` to the app's **Secrets** in the Streamlit dashboard.
